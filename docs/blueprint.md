# **App Name**: Caçambas Brasil - Aluguel Simplificado

## Core Features:

- Banner Principal Interativo: Exibir um banner chamativo com a frase 'Aluguel de Caçambas Rápido e Seguro' e um botão de destaque para 'Solicitar Orçamento via WhatsApp'.
- Listagem de Serviços: Apresentar de forma clara os serviços oferecidos: aluguel de caçambas para entulho, obras e reformas, limpeza de terrenos, atendimento a empresas e condomínios.
- Informações da Empresa: Disponibilizar uma breve apresentação da Caçambas Brasil, destacando o compromisso com pontualidade, sustentabilidade e atendimento rápido.
- Contato Direto via WhatsApp: Implementar um botão fixo e flutuante do WhatsApp para facilitar o contato imediato dos clientes.
- Formulário de Contato Simplificado: Criar um formulário de contato simples com campos para nome, telefone e mensagem, garantindo facilidade no envio de solicitações.
- Otimização de SEO: Implementar SEO básico para otimizar o site para buscas como 'aluguel de caçamba perto de mim'.

## Style Guidelines:

- Primary color: Strong industrial yellow (#FFC107), conveys energy and reliability.
- Background color: Light yellow (#FAF9F0), provides a neutral backdrop.
- Accent color: Industrial orange (#FF5722), used for call-to-action buttons and important highlights.
- Body font: 'PT Sans', a modern sans-serif to ensure readability and accessibility.
- Headline font: 'Space Grotesk', sans-serif and techy for headlines
- Use simple icons related to construction sites and bins
- Visual limpo, moderno e profissional.
