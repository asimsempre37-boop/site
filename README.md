
# Prime Caçamba

Este é o site oficial da **Prime Caçamba**, especializado em aluguel de caçambas para entulho, obras e limpeza urbana.

## Tecnologias Utilizadas

- **Next.js 15 (App Router)**
- **Tailwind CSS** para estilização
- **Shadcn UI** para componentes de interface
- **Lucide React** para ícones
- **Genkit** para integrações de IA futura

## Como começar

1. O código principal da página está em `src/app/page.tsx`.
2. Os componentes globais estão em `src/components/layout/`.
3. Para alterações visuais, consulte `src/app/globals.css`.
